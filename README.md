# gentlerain.ai – Frontend Clone

This project is a front-end clone inspired by the gentlerain.ai landing page.  
It was implemented in multiple stages to progressively enhance structure, styling, and interactivity using modern web technologies.

---

## Task 1 – Base Structure & Layout

Task 1 focused on setting up a clean and scalable foundation for the project.

### Features Implemented
- Semantic HTML structure for clear layout and accessibility
- Base CSS styling for layout, spacing, and typography
- Fixed navigation header
- Hero section with basic parallax effect
- Responsive layout for smaller screen sizes

### Technologies Used
- HTML5
- CSS3
- JavaScript (Vanilla)

---

## Task 2 – Animations & Interactions

Task 2 focused on enhancing the user experience through animations and interactive UI components.

### Features Implemented
- **Multi-layer Parallax**
  - Layered parallax effect in the hero section to create visual depth during scroll

- **Scroll-based Animations**
  - Smooth scroll-triggered animations using **AOS (Animate On Scroll)**

- **Interactive Flip Cards**
  - 3D flip cards implemented using pure CSS transforms to reveal additional content on hover

- **Text Animations**
  - Subtle entrance animations for hero and section text to improve visual flow

### Technologies Used
- HTML5
- CSS3 (Flexbox, animations, 3D transforms)
- JavaScript (Vanilla)
- AOS (Animate On Scroll)

---

## Approach

The project emphasizes clean structure, performance, and maintainability.  
Animations are kept subtle and purposeful to enhance usability without overloading the interface.

---

## Status
